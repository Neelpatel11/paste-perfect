# Lexical Example

This example shows how to integrate Paste Perfect with [Lexical](https://lexical.dev/), Facebook's extensible text editor.

## Setup

```bash
npm install
npm run dev
```

## Integration

The example demonstrates:

1. Using `cleanPaste` with Lexical's `onPaste` handler
2. Cleaning HTML and converting to Lexical nodes
3. Maintaining Lexical's structure and formatting
