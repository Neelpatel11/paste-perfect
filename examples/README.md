# Paste Perfect Examples

This directory contains example integrations of Paste Perfect with various rich text editors.

## Examples

- **[contenteditable](./contenteditable/)** - Basic contenteditable integration
- **[Tiptap](./tiptap/)** - Integration with [Tiptap](https://tiptap.dev/)
- **[Lexical](./lexical/)** - Integration with [Lexical](https://lexical.dev/)

## Running Examples

Each example has its own setup instructions. See the README in each example directory.

Generally:

```bash
cd examples/tiptap  # or lexical, contenteditable
npm install
npm run dev
```

## Adding New Examples

We welcome new example integrations! If you've integrated Paste Perfect with another editor, feel free to add it here.
