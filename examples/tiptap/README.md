# Tiptap Example

This example shows how to integrate Paste Perfect with [Tiptap](https://tiptap.dev/), a headless rich text editor.

## Setup

```bash
npm install
npm run dev
```

## Integration

The example demonstrates:

1. Using `cleanPaste` with Tiptap's `onPaste` handler
2. Cleaning HTML before inserting into the editor
3. Maintaining Tiptap's formatting and structure
